# Публичный Evaluation Kit

## Лучший путь для review

1. Прочитать `README_RU.md`
2. Прочитать `KNOWN_LIMITATIONS_RU.md`
3. Прочитать `EVALUATE_THIS_REPO_RU.md`
4. Посмотреть `docs/ru/operational-retries.md`
5. Посмотреть `docs/ru/scheduled-operations.md`
6. Открыть validator page на docs-site
7. Запустить tests и smoke checks

## Важное предупреждение

Некоторые слои проекта уже production-trustworthy, а некоторые пока starter-grade.
Оценивайте текущее состояние, а не воображаемый hosted product.
