# Self-Hosted Use Cases

Платформа бесплатная, прозрачная и self-hosted first.

## Для себя

- запускать discoverability-аудиты на своих сайтах
- хранить все данные в собственной инфраструктуре
- адаптировать prompts, checklists, reports и свои cloud/local AI providers

## Для команды

- стандартизировать SEO, GEO и AI-audit workflows
- сохранять exportable artifacts
- избегать mandatory vendor lock-in
- приглашать editors, admins и viewers в общие workspaces

## Для агентства

- развернуть платформу для внутренних операторов
- использовать white-label заголовки workspace и report framing
- адаптировать приложение под client-facing internal workflows
- подключать client-specific providers или local LLM environments

## Для собственного продукта или сервиса

- форкнуть
- расширить
- построить поверх бесплатный или коммерческий сервис
- сохранить контроль над deployment и данными
