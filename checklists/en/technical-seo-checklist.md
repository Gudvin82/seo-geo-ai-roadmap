# Technical Seo Checklist

Use this as a quick execution card, not as the full methodology.

Deep companion:

- [docs/en/technical-seo-deep-playbook.md](../../docs/en/technical-seo-deep-playbook.md)

## Crawl and indexability

- [ ] Indexable pages return `200`
- [ ] Important pages are not blocked in `robots.txt`
- [ ] Sitemap is clean, current, and canonical
- [ ] Redirects do not create chains

## Rendering and page meaning

- [ ] Server-rendered HTML is available for critical pages
- [ ] Title, H1, and key commercial meaning are visible in raw HTML
- [ ] Canonical rules are intentional

## Architecture and UX

- [ ] Internal links cover money pages
- [ ] Core Web Vitals reviewed on high-intent pages

## Proof

- [ ] Before/after evidence saved
- [ ] Owner and validation rule assigned
