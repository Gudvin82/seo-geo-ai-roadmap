# ROI Calculation Example

## EN

Input assumptions:

- Traffic: 5,000
- Conversion rate: 3%
- Lead-to-sale rate: 20%
- Average check: 1,200
- Margin rate: 45%
- SEO cost: 1,500
- AI-referred share: 10%
- Period: monthly

Resulting metrics:

- Visits: 5,000
- Leads: 150
- Sales: 30
- Revenue: 36,000
- Gross margin: 16,200
- Cost: 1,500
- Estimated ROI / ROMI: 980%

Interpretation:

This scenario suggests a healthy return profile, but the result depends heavily
on conversion quality and real margin assumptions.
