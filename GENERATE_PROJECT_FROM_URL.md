# Generate Project From URL

If a user gives you a site URL and asks you to build with this platform:

1. detect project type
2. detect market mode
3. propose the best integration bundle
4. generate a manifest through the project-generation contract
5. produce setup docs, secrets checklist, and operator handoff

Required output:

- blueprint JSON
- integration checklist
- starter UI shell
- deployment mode recommendation
- verified vs assumed states
