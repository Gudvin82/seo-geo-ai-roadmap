# Чем Этот Проект Не Является

Этот проект не стоит описывать как:

- hosted SaaS, который поддерживает сам автор
- zero-setup enterprise scanner с уже живыми SLA и billing
- магический SEO black box с гарантированными позициями
- движок с гарантированными AI citations
- полностью автономный ИИ, который тихо правит production-сайты без approval

## Важные границы

- Часть integrations уже структурирована как production-ready path, но все еще
  остается под контролем оператора.
- Репозиторий по замыслу бесплатный и self-hosted.
- В коде есть SaaS-подобные слои, но нет публичного hosted tier от автора.
- Поддержка AI-агентов реальна, но approval-first governance намеренно сохранен.

## Честное обещание

Честное обещание проекта такое:

- можно использовать вручную
- можно отдать AI coding agent
- можно self-hosted развернуть и брендировать под себя

Честное обещание проекта не такое:

- "зарегистрируйся по публичной ссылке и получи готовый коммерческий SaaS от автора"
