# SEO + GEO + AI Продвижение

Открытая карта продвижения сайта для классического SEO, локального GEO, AI-видимости и двуязычных контент-операций.

[English version](./README.md)

## Что внутри

- 25 разделов roadmap на русском и английском
- Чеклисты для запуска, контента, local SEO, AI/GEO и технического SEO
- Шаблоны `llms.txt`, `robots.txt`, schema, брифов, KPI-дашборда и RACI
- Готовые промпты для семантики, AI-аудита, контент-брифов, E-E-A-T и анализа конкурентов
- Стартовые Python-скрипты для проверки AI-ботов, `llms.txt`, sitemap и schema

## Для кого

- SEO-специалисты и in-house команды
- Фаундеры, CMO и growth-команды
- Агентства, работающие с Google, Яндексом, Bing и AI-поиском
- Контент- и GEO-команды, которым нужна повторяемая система

## Быстрый старт

1. Начните с [docs/ru/00-business-strategy.md](./docs/ru/00-business-strategy.md).
2. Пройдите аудит из [docs/ru/01-audit.md](./docs/ru/01-audit.md).
3. Приоритизируйте AI-видимость через [docs/ru/08-geo-ai-search.md](./docs/ru/08-geo-ai-search.md).
4. Возьмите шаблоны из [templates](./templates/).
5. Используйте промпты из [prompts/ru](./prompts/ru/).

## Структура репозитория

- `docs/en`, `docs/ru`: двуязычная карта
- `checklists/`: исполнительские чеклисты
- `templates/`: переиспользуемые шаблоны
- `prompts/`: AI-промпты
- `scripts/`: утилиты проверки
- `examples/`: заполненные примеры
