# What This Project Is Not

This project should not be described as:

- a maintainer-operated hosted SaaS
- a zero-setup enterprise scanner with SLA and billing already live
- a magic SEO black box with guaranteed rankings
- a guaranteed AI citation engine
- a fully autonomous AI that silently edits production sites without approval

## Important boundaries

- Some integrations are production-structured but still operator-owned.
- The repo is free and self-hosted by design.
- The platform includes SaaS-like layers in code, but not a public hosted tier
  run by the maintainer.
- AI-agent support is real, but approval-first governance is intentional.

## Honest promise

The honest promise is:

- you can use it manually
- you can give it to an AI coding agent
- you can self-host and brand it as your own platform

The honest promise is not:

- "sign up on a public cloud URL and get a finished commercial SaaS from the
  maintainer"
