# Research → Build → Improve → Re-measure

This is the preferred `v3.8.0` operating loop.

## Step 1. Research

- `/geo quick` for pre-sales or triage
- `/geo audit` for a full baseline
- `/geo brands` for truth-center drift
- `/geo platforms` for AI visibility differences

## Step 2. Explain

- `/geo graph` to show issue relationships
- `/geo report` to translate findings into executive and client language

## Step 3. Improve

- `/geo technical`
- `/geo schema`
- `/geo content`
- `/geo citability`

## Step 4. Re-measure

- `/geo compare`
- scheduled checks
- SoV snapshots
- report history

## Best fit

This loop works for agencies, in-house teams, founders, and consultants because
it turns isolated audits into a repeatable system.
