# White-Label Mode

Use workspace branding fields when you need a client-facing layer on top of the
same shared platform.

## Current white-label fields

- `client_report_title`
- `client_report_subtitle`
- `client_report_footer`
- `branding_logo_url`

## Recommended workflow

1. Create one workspace per client.
2. Fill white-label fields before exporting reports.
3. Keep one brand-facts profile per site or per business line.
4. Export artifacts only after human review.
