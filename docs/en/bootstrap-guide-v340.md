# Bootstrap Guide

Use the bootstrap planner when you want the fastest safe path into the project.

## Demo mode

```bash
python scripts/bootstrap_self_hosted.py --mode demo --format markdown
```

## Production-like mode

```bash
python scripts/bootstrap_self_hosted.py --mode production --format markdown
```

This is not a one-click hosted installer. It is a practical self-hosted
bootstrap guide that stays honest about what the operator still needs to do.
