# GEO Meta-Case: This Repository

## Why include this

The repository itself is a GEO/AI test case. It has public docs, public
templates, a self-hosted app layer, and AI-agent handoff instructions.

## What we do for its discoverability

- bilingual EN/RU documentation
- transparent methodology and proof assets
- `llms.txt` support and AI bot guidance
- repeatable release notes and changelog discipline

## Honest boundary

We can monitor its discoverability patterns, but we do not claim stable AI
citations for the repo itself.
