# Definition of Done

> **Tags:** [U]
> **Priority:** Medium
> **Roadmap phase:** Month 0-1

## Why this matters

DoD prevents teams from calling a task complete before it is shipped, indexed, and measurable.

## Step-by-step

1. Define required checks for each work type.
2. Include QA, tracking, and rollback notes.
3. Require indexability and internal links for new pages.
4. Review DoD after each incident.
