# v5.3.0 Release Summary

`v5.3.0` strengthens the social intelligence and SaaS delivery layers of the
platform.

Highlights:

- new starter integrations for X Ads, X Organic, Threads, Reddit mentions, and
  TikTok organic
- a social-intelligence center that turns post performance, community mentions,
  and repeated questions into useful content and demand opportunities
- a stronger SaaS growth center with service tiers, client surfaces, and growth
  loops for agencies, operators, and product teams
- a public project image asset added to the repository and docs site

This release is about helping teams do more than measure search. It helps them:

- map supporting social demand
- parse useful community and creator signals
- convert those signals into landing, FAQ, and proof improvements
- package the platform more clearly as a client-facing self-hosted service
