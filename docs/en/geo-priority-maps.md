# GEO Priority Maps

## Priority matrix by site type

| Site type | SEO first | GEO/AI first | Conversion first |
|---|---|---|---|
| Local business | local pages, NAP, maps, reviews | local entity clarity, FAQ, bot access | trust blocks, calls, directions, proof |
| B2B services | service architecture, intent mapping | fact consistency, expert proof, answer-ready service pages | case studies, forms, CTA sequencing |
| SaaS | technical SEO, docs, product pages | structured facts, use-case pages, llms.txt | product proof, onboarding CTA, pricing clarity |
| Media/content | taxonomy, freshness, internal links | summary blocks, citation-ready facts, source clarity | newsletter, subscription, authority loops |
| Personal brand | entity hub, about, talk pages | biography consistency, interviews, citations | consultation CTA, trust, proof |
| E-commerce | category/indexation, product schema | product/entity consistency, FAQ, policy clarity | shipping, returns, reviews, offer clarity |

## Minimal decision rule

- If the site sells trust first, prioritize facts, proof, and human clarity.
- If the site sells volume first, prioritize technical scale and structured
  content.
- If the site is multilingual, prioritize entity governance before expansion.
