# v5.2.0 Release Summary

`v5.2.0` turns the platform into a more understandable and more deployable
product layer.

Highlights:

- repo-understanding mode for AI agents and operators
- deploy wizard for local, VPS Docker, Coolify, Railway, Render, and Kubernetes
- integration health center for sync freshness, retries, diagnostics, and next
  steps
- portfolio dashboard, demo center, and productization center
- proof export pack, mention or reputation center, and operator board
- broader AI-to-App templates: landing page, scanner SaaS, agency workspace,
  local business dashboard, and e-commerce ops

This release is about reducing friction between:

- understanding the repo
- deploying the product
- operating multiple projects
- proving outcomes
- handing the platform to an AI coding agent with less custom prompting
