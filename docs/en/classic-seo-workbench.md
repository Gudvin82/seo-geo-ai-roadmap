# Classic SEO Workbench

`v6.9.0` strengthens the classical SEO layer so it stands next to GEO, not
behind it.

## Workbench tracks

- semantic execution
- competitor workflows
- authority and link depth
- benchmark datasets

## Scripts

- `scripts/semantic_gap_mapper.py`
- `scripts/serp_competitor_matrix.py`
- `scripts/link_gap_summary.py`
- `scripts/benchmark_dataset_builder.py`

## Operator rule

Treat semantics, competitors, links, and benchmarks as one operating system,
not four unrelated tasks.
