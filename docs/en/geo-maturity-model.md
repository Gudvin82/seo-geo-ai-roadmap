# GEO Maturity Model

## Levels

| Level | State | What is true |
|---|---|---|
| 1 | Basic SEO hygiene | site is crawlable but GEO work is mostly absent |
| 2 | Entity and fact consistency | core claims, names, and proof are normalized |
| 3 | AI-ready structured content | answer-ready pages and schema exist on key surfaces |
| 4 | Monitored AI visibility | citations and answer surfaces are tracked as proxies |
| 5 | Governed multi-market system | discoverability is managed across languages, markets, and teams |

## Upgrade path

- 1 → 2: fix canonical facts, about, trust, and entity naming
- 2 → 3: publish answer-ready pages and structured data
- 3 → 4: add AI visibility baselines, prompt tracking, and reporting
- 4 → 5: govern multilingual and multi-market execution with explicit owners

## Minimal program when resources are tight

- Level 1-2 work on money pages first
- one answer-ready FAQ or service hub
- one visibility baseline
- one owner for fact governance
