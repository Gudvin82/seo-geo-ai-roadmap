# llms.txt Validator

## What it is

`v3.2.0` adds a free validator asset for `llms.txt`.

Use:

- API endpoint: `POST /api/v1/tools/llms-validator`
- standalone page: [`app/frontend/llms-validator.html`](../../app/frontend/llms-validator.html)

## Input modes

- public URL to `llms.txt`
- pasted content

## Output

- validity signal
- warnings
- recommendations
- observed facts about structure and coverage
