# Yandex + LLM Dual Optimization Playbook

## Goal

Improve both Yandex discoverability and LLM answer readiness without creating
two conflicting site versions.

## Steps

1. Fix Yandex basics: region, commercial factors, local trust, technical health.
2. Align RU fact pages, offer wording, and FAQ blocks.
3. Publish answer-ready RU sections and structured data.
4. Track Yandex metrics separately from AI visibility proxies.
5. Keep one source of truth for facts across both layers.
