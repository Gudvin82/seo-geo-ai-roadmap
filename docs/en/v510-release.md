# v5.1.0 Release Summary

`v5.1.0` is the operating-center, managed-runtime maturity, and AI-buildable
project generation release.

## What changed

- Deepened the executive dashboard into an operating center with weekly
  narrative, anomaly detection, owner suggestions, benchmark overlays, and
  portfolio-style summary fields.
- Added SaaS-box UI and API support for workspace catalog views, tenant
  overview routing, and organization-plus-tenant management surfaces.
- Added a proof timeline that combines evidence records, experiments, audit
  runs, reports, and artifacts into one project-level causality stream.
- Upgraded AI-to-App from manifest-only planning to scaffold generation that
  writes starter project files under `generated_projects/`.
- Added one-link builder, proof kit, and social-distribution center entrypoints
  to make AI handoff and operator setup more turnkey.

## Main outcome

The repository is now stronger not only as a methodology and self-hosted app,
but as a platform an external AI can use to:

1. understand the contract
2. generate a working scaffold
3. connect the right integrations
4. produce executive and proof surfaces
5. hand the result back with less operator friction
