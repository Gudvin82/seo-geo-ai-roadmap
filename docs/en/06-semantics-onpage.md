# Semantics and On-Page

> **Tags:** [U]
> **Priority:** High
> **Roadmap phase:** Month 1-3

## Why this matters

Semantic clustering ensures each page serves one dominant intent instead of competing with itself.

## Step-by-step

1. Build keyword clusters by intent.
2. Map one cluster to one page.
3. Optimize title, H1, intro, FAQs, and internal anchors.
4. Remove cannibalization and page overlap.
5. Refresh content after ranking data returns.
