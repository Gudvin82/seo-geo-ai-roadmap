# Technical SEO

> **Tags:** [U]
> **Priority:** High
> **Roadmap phase:** Month 0-2

## Why this matters

Technical debt blocks indexing, slows rendering, and weakens both human and AI access.

## Step-by-step

1. Validate indexation, canonicals, redirects, and robots directives.
2. Review XML sitemaps and orphan pages.
3. Improve CWV, HTML availability, and mobile stability.
4. Fix duplication, pagination, and crawl traps.
5. Re-test after each release.
