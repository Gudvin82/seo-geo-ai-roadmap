# Managed Runtime Proof

`v6.9.0` upgrades the integration layer from a generic managed-runtime matrix to
a more operator-grade proof surface.

## What this adds

- explicit auth model per first-class integration
- refresh lifecycle and cadence clarity
- failure signals and recovery playbooks
- proof assets for screenshots, exports, and demo evidence

## Best use

Use this layer when you need to explain whether an integration is merely
connected, truly reviewable, or ready to support executive reporting.
