# v6.2.0 Release Summary

`v6.2.0` turns the repo into a stronger operating platform by adding a
first-class SEO intelligence layer on top of the existing SEO, GEO, AI, RU, and
self-hosted foundations.

## What changed

- added native SEO intelligence starter surfaces for:
  - keyword research
  - competitor intelligence
  - backlink intelligence
  - rank tracking
- added `/api/v1/settings/seo-intelligence-center`
- expanded the executive dashboard with SEO intelligence signals and anomalies
- added version-consistency checks for release hygiene
- strengthened docs routing with clearer "what this project is / is not"
  positioning
- updated the repo and CI surface so version drift is caught automatically

## Why it matters

Before `v6.2.0`, the repo was strong at:

- methodology
- GEO/AI framing
- RU/Yandex execution
- self-hosted app and scanner flows

After `v6.2.0`, the product can also express a clearer in-repo SEO intelligence
loop:

- demand map
- competitor gaps
- authority and backlink recovery
- tracked rank visibility

## Honest boundary

`v6.2.0` still does **not** mean:

- a maintainer-operated hosted SaaS
- guaranteed rankings or guaranteed AI citations
- a black-box enterprise SEO suite with every external dataset bundled by default

It does mean the repo is a stronger self-hosted platform and a stronger AI-agent
handoff surface.
