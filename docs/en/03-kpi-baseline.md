## Why this matters

Baseline turns SEO from opinion into operations. It defines what changes after implementation.

1. Record traffic, rankings, leads, sales, and branded demand.
2. Add crawl health, Core Web Vitals, and index coverage.
3. Add AI visibility metrics and citation share.
4. Set 3, 6, and 12 month targets.
5. Publish a shared dashboard.
