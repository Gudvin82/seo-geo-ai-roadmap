# Bootstrap Guide

Используйте bootstrap planner, когда нужен самый быстрый и при этом безопасный
путь входа в проект.

## Demo mode

```bash
python scripts/bootstrap_self_hosted.py --mode demo --format markdown
```

## Production-like mode

```bash
python scripts/bootstrap_self_hosted.py --mode production --format markdown
```

Это не one-click hosted installer. Это practical self-hosted bootstrap guide,
который честно показывает, что оператору все еще нужно сделать руками.
