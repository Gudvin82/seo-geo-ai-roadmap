# GEO Meta-Case: этот репозиторий

## Зачем это включать

Сам репозиторий уже является GEO/AI test case. У него есть public docs, public
templates, self-hosted app layer и AI-agent handoff instructions.

## Что мы делаем для его discoverability

- двуязычная EN/RU-документация
- прозрачная методология и proof assets
- поддержка `llms.txt` и guidance по AI-ботам
- repeatable release notes и changelog discipline

## Честная граница

Мы можем наблюдать его discoverability patterns, но не обещаем стабильные AI
citations для самого репозитория.
