# Multilingual Entity Governance Playbook

## Цель

Сохранить одну coherent brand entity на нескольких языках и рынках.

## Шаги

1. Определите canonical global facts.
2. Определите допустимые local adaptations.
3. Сравните homepage, market pages и local profile pages.
4. Синхронизируйте structured data, hreflang и формулировки facts.
5. Проверяйте market-specific AI и search surfaces отдельно.
