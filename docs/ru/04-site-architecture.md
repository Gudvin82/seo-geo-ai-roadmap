# Архитектура сайта

> **Tags:** [U] [INT]
> **Priority:** High
> **Roadmap phase:** Месяц 0-1

## Зачем это нужно

Архитектура определяет глубину обхода, тематическую ясность и соответствие страниц поисковому интенту.

## Пошагово

1. Определите обязательные типы страниц.
2. Сгруппируйте страницы по темам, гео, индустрии и этапу воронки.
3. Сделайте URL-логику предсказуемой и масштабируемой.
4. Простройте внутреннюю перелинковку через parent-child связи.
