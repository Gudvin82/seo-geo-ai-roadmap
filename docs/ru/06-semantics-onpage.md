# Семантика и on-page

> **Tags:** [U]
> **Priority:** High
> **Roadmap phase:** Месяц 1-3

## Зачем это нужно

Кластеризация семантики помогает каждой странице закрывать один доминирующий интент, а не конкурировать с соседними.

## Пошагово

1. Соберите кластеры ключей по интенту.
2. Сопоставьте один кластер одной странице.
3. Оптимизируйте title, H1, интро, FAQ и анкоры.
4. Уберите каннибализацию и пересечения.
