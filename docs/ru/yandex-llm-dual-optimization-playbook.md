# Yandex + LLM Dual Optimization Playbook

## Цель

Улучшить и Yandex discoverability, и LLM answer readiness, не создавая две
конфликтующие версии сайта.

## Шаги

1. Закройте Yandex basics: региональность, commercial factors, local trust,
   technical health.
2. Синхронизируйте RU fact pages, offer wording и FAQ blocks.
3. Публикуйте answer-ready RU sections и structured data.
4. Отслеживайте Yandex metrics отдельно от AI visibility proxies.
5. Держите один source of truth для facts на оба слоя.
