# Classic SEO Workbench

`v6.9.0` усиливает classical SEO layer так, чтобы он стоял рядом с GEO, а не
отставал от него.

## Треки workbench

- semantic execution
- competitor workflows
- authority and link depth
- benchmark datasets

## Scripts

- `scripts/semantic_gap_mapper.py`
- `scripts/serp_competitor_matrix.py`
- `scripts/link_gap_summary.py`
- `scripts/benchmark_dataset_builder.py`

## Operator rule

Трактуйте semantics, competitors, links и benchmarks как одну operating system,
а не как четыре несвязанных задачи.
