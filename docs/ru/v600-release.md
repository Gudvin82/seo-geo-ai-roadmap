# Сводка релиза v6.0.0

`v6.0.0` — это релиз на trust density.

Он пытается выиграть не за счет еще большего surface area, а за счет
сокращения разрыва между упаковкой и содержанием.

## Что изменилось

- новое корневое объяснение методологии
- прозрачное объяснение scoring с привязкой к реальным code paths
- новые AI task packs для audit, deployment, improvement, productization и public evaluation
- более глубокие практические playbooks по:
  - technical SEO
  - semantic core и intent
  - competitor и authority gaps
  - GEO и AI operations
- обновленные core checklists, которые теперь прямо позиционируются как quick execution cards, а не как вся методология
- более чистая docs routing через current entrypoints и явную archive policy
- новый bounded public audit case для `anmalishev.ru`

## Почему этот релиз важен

До `v6.0.0` repo был сильным, но его было легче считать:

- overpackaged
- слишком нагруженным release-notes слоем
- более впечатляющим по breadth, чем по density

`v6.0.0` прямо отвечает на это восприятие и делает:

- core path более ясным
- scoring более объяснимым
- playbooks более практичными
- AI handoff более конкретным
- evidence layer более видимой

## Граница не изменилась

`v6.0.0` все еще **не** заявляет:

- hosted SaaS от автора
- идеальный или универсальный scoring
- законченный enterprise SEO-manual по всем поддисциплинам
- автономные production edits без review
