# Research → Build → Improve → Re-measure

Это рекомендуемый `v3.8.0` operating loop.

## Шаг 1. Исследовать

- `/geo quick` для pre-sales или triage
- `/geo audit` для полного baseline
- `/geo brands` для проверки truth-center drift
- `/geo platforms` для различий по AI visibility

## Шаг 2. Объяснить

- `/geo graph`, чтобы показать связи между проблемами
- `/geo report`, чтобы перевести findings в executive и client language

## Шаг 3. Улучшить

- `/geo technical`
- `/geo schema`
- `/geo content`
- `/geo citability`

## Шаг 4. Перемерить

- `/geo compare`
- scheduled checks
- SoV snapshots
- история отчетов

## Кому подходит

Этот цикл подходит агентствам, in-house командам, фаундерам и консультантам,
потому что превращает разрозненные аудиты в повторяемую систему.
