# Сводка релиза v5.1.0

`v5.1.0` — это релиз про operating center, managed-runtime maturity и
AI-buildable project generation.

## Что изменилось

- Executive dashboard усилен до operating center: weekly narrative, anomaly
  detection, owner suggestions, benchmark overlays и portfolio-style summary.
- Добавлены SaaS-box UI и API surfaces для workspace catalog, tenant overview и
  organization-plus-tenant management.
- Добавлен proof timeline, который собирает evidence records, experiments,
  audit runs, reports и artifacts в единый causality stream по проекту.
- AI-to-App поднят с manifest-only planning до scaffold generation, который
  реально пишет starter files в `generated_projects/`.
- Добавлены one-link builder, proof kit и social-distribution center, чтобы AI
  handoff и operator setup были ближе к turnkey-сценарию.

## Главный результат

Теперь репозиторий сильнее не только как методология и self-hosted app, но и
как платформа, которую внешний ИИ может использовать, чтобы:

1. понять контракт
2. сгенерировать рабочий scaffold
3. подключить нужные интеграции
4. собрать executive и proof surfaces
5. вернуть результат оператору с меньшим трением
