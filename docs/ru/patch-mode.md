# Patch Mode

Patch mode переводит платформу из режима "найти проблему" в режим
"подготовить внедрение".

## Типы artifacts

- issue-ready markdown
- developer task brief
- content brief
- suggestion для schema или `llms.txt`
- draft FAQ или answer-ready copy

## Текущее поведение

- outputs оформляются как явные artifacts
- human review остается обязательным
- CMS writeback в `v2.2.0` не автоматический
