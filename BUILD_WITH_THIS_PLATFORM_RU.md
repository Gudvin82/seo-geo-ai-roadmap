# Строить С Этой Платформой

Используйте этот репозиторий как платформу, а не только как набор документов.

## Что внешний ИИ должен сделать

1. Прочитать `README_RU.md`, `START_HERE_FOR_AI_RU.md` и `EVALUATE_THIS_REPO_FIRST_RU.md`
2. Прочитать generation contracts в `contracts/*.schema.json`
3. Определить, что нужно:
   - self-hosted
   - SaaS box
   - managed deployment
4. Задать только недостающие вопросы
5. Сгенерировать полный project instance:
   - site shell
   - admin shell
   - scanner
   - integrations
   - executive dashboard
   - tasks и handoff docs

## Правила

- отделять verified state от assumed state
- держать EN/RU поддержку синхронной
- предпочитать machine-readable outputs, а не только prose-планы
