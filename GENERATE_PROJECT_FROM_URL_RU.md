# Сгенерировать Проект Из URL

Если пользователь дает URL сайта и просит строить проект через эту платформу:

1. определить project type
2. определить market mode
3. предложить лучший integration bundle
4. сгенерировать manifest через project-generation contract
5. подготовить setup docs, secrets checklist и operator handoff

Обязательный output:

- blueprint JSON
- integration checklist
- starter UI shell
- рекомендация по deployment mode
- verified vs assumed states
