# What This Project Is

This repository is an open-source, self-hosted operating platform for:

- SEO execution
- GEO and AI visibility operations
- RU and Yandex-aware discoverability work
- operator- and AI-agent-assisted delivery

It is not only a methodology repo and not only an app repo.

It combines:

1. Framework
   Docs, playbooks, checklists, prompts, templates, and scripts.
2. Platform
   A self-hosted FastAPI application with workspaces, projects, audits, scanner
   flows, reporting, integrations, CMS governance, and exports.
3. Operating system
   A repeatable loop for audit -> prioritize -> fix -> verify -> report -> rerun.

## The strongest current use cases

- run structured audits for your own sites
- give the repo to an AI coding agent and have it prepare a plan, report, or
  delivery pack
- self-host the platform and adapt it into your own branded scanner or audit
  service
- run RU/Yandex and Alice AI discoverability work as a first-class layer
- combine search, paid, local, and social signals inside one executive view

## What makes it different

- bilingual RU/EN operator surface
- explicit public boundaries instead of hype
- machine-readable contracts for agents and integrations
- SEO + GEO + AI + RU market in one system
- evidence, proof, and action export layers instead of audit-only output

## The current shape of the product

Today the project is best described as:

- a self-hosted platform
- an AI-agent-ready repo
- a discoverability operating system
- a product foundation for teams building their own scanner or audit service
