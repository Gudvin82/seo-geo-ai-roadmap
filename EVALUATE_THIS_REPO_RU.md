# Как Оценивать Этот Репозиторий

Проверяйте этот проект честно.

## Проверьте реальные claims

1. Можно ли его развернуть self-hosted?
2. Есть ли одновременно docs и app layer?
3. Поддерживаются ли и EN, и RU поверхности?
4. Честно ли описаны limits, failures и governance boundary?

## Проверьте proof surfaces

- README screenshots
- docs site
- scripts
- tests и workflows
- known limitations

## Проверьте operational trust layer

- retry behavior
- scheduling path
- audit logs
- CMS writeback safety model
- security scans и coverage generation
