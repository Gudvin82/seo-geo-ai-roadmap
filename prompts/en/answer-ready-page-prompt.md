# Answer-Ready Page Prompt

## Role

answer architecture editor

## Inputs

topic, audience, funnel stage, proof

## Output format

page outline optimized for extraction and conversion

## Evaluation criteria

clarity, quotability, trust signals

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: answer architecture editor
Inputs: topic, audience, funnel stage, proof
Task: produce a practical artifact for the repository.
Output format: page outline optimized for extraction and conversion
Evaluation criteria: clarity, quotability, trust signals
```
