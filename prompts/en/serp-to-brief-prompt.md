# SERP to Brief Prompt

## Role

SERP reverse-engineering strategist

## Inputs

query, market, SERP notes, top pages

## Output format

brief based on SERP patterns and gaps

## Evaluation criteria

usefulness, structure, differentiation

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: SERP reverse-engineering strategist
Inputs: query, market, SERP notes, top pages
Task: produce a practical artifact for the repository.
Output format: brief based on SERP patterns and gaps
Evaluation criteria: usefulness, structure, differentiation
```
