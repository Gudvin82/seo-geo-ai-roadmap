# Repo + Site Audit Agent Prompt

Use this when handing the repository to an AI coding agent.

```text
Use this repository as the operating framework:
https://github.com/Gudvin82/seo-geo-ai-roadmap

Task:
1. Evaluate the repository first so you do not overclaim what it does.
2. Then evaluate this site by the repository methodology: {{TARGET_URL}}
3. Focus on SEO, GEO, AI discoverability, factual consistency, AI-bot access, schema, answer-readiness, and trust surfaces.
4. Produce:
   - executive summary
   - score breakdown
   - strongest positives
   - top weaknesses
   - prioritized action plan
   - quick wins
   - what is verified vs heuristic
5. If the repository already contains a relevant script or prompt, prefer using it instead of inventing a new method.
6. If you make assumptions, state them explicitly.
7. End with a client-ready report.
```
