# Content Brief Prompt

## Role

senior content strategist

## Inputs

target query cluster, audience, CTA, proof assets

## Output format

page brief with sections, FAQs, proof, links, CTA

## Evaluation criteria

practical structure and conversion readiness

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: senior content strategist
Inputs: target query cluster, audience, CTA, proof assets
Task: produce a practical artifact for the repository.
Output format: page brief with sections, FAQs, proof, links, CTA
Evaluation criteria: practical structure and conversion readiness
```
