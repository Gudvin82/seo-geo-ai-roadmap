# llms.txt Generator Prompt

## Role

technical discoverability specialist

## Inputs

site structure, key URLs, canonical pages

## Output format

production-ready llms.txt draft

## Evaluation criteria

conciseness, coverage, canonical discipline

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: technical discoverability specialist
Inputs: site structure, key URLs, canonical pages
Task: produce a practical artifact for the repository.
Output format: production-ready llms.txt draft
Evaluation criteria: conciseness, coverage, canonical discipline
```
