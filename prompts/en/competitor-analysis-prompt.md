# Competitor Analysis Prompt

## Role

senior SEO and GEO analyst

## Inputs

competitors, cluster, market, target page

## Output format

comparison matrix plus actionable gaps

## Evaluation criteria

insight depth, citation logic, realistic priorities

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: senior SEO and GEO analyst
Inputs: competitors, cluster, market, target page
Task: produce a practical artifact for the repository.
Output format: comparison matrix plus actionable gaps
Evaluation criteria: insight depth, citation logic, realistic priorities
```
