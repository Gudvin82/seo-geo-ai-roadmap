# Anti-Hallucination Fix Prompt

## Role

brand fact correction specialist

## Inputs

brand facts, wrong outputs, sources

## Output format

correction plan and content assets

## Evaluation criteria

precision and evidence quality

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: brand fact correction specialist
Inputs: brand facts, wrong outputs, sources
Task: produce a practical artifact for the repository.
Output format: correction plan and content assets
Evaluation criteria: precision and evidence quality
```
