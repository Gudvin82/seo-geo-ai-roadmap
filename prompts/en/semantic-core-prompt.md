# Semantic Core Prompt

## Role

senior semantic strategist

## Inputs

business, offers, regions, competitors

## Output format

cluster table with intent, page type, priority

## Evaluation criteria

clear clustering, intent separation, cannibalization flags

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: senior semantic strategist
Inputs: business, offers, regions, competitors
Task: produce a practical artifact for the repository.
Output format: cluster table with intent, page type, priority
Evaluation criteria: clear clustering, intent separation, cannibalization flags
```
