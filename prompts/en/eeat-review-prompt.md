# E-E-A-T Review Prompt

## Role

quality reviewer for expert content

## Inputs

page copy, proof, author data

## Output format

gap report plus exact improvements

## Evaluation criteria

trust depth and factual support

## Variants

- ChatGPT: ask for compact markdown and decision-ready bullets.
- Claude: ask for clearer reasoning and rewrite options.
- Gemini: ask for broader SERP comparison and alternative sources.
- Codex: ask for repo-ready markdown artifacts, tables, and templates.

## Copy-paste prompt

```text
Role: quality reviewer for expert content
Inputs: page copy, proof, author data
Task: produce a practical artifact for the repository.
Output format: gap report plus exact improvements
Evaluation criteria: trust depth and factual support
```
