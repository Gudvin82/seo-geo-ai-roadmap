# Пакет промптов для AI-аудита

## 1. Проверь мой бренд в ChatGPT

```text
Ты проводишь аудит AI-видимости. Для бренда [X] опиши, как ChatGPT, вероятно, его представит, каких фактов может не хватать и какие источники нужно усилить.
```

## 2. Найди конкурентов, которых рекомендует Perplexity вместо меня

```text
Для темы [Y] определи, какие компании Perplexity, вероятно, порекомендует вместо бренда [X], и объясни почему.
```

## 3. Сгенерируй llms.txt для сайта [URL]

```text
Сгенерируй llms.txt для [URL]. Сохрани краткость, перечисли важнейшие разделы и используй только canonical URL.
```

## 4. Проверь галлюцинации о бренде [X]

```text
Перечисли вероятные галлюцинации или фактические неоднозначности, которые LLM может выдать о бренде [X], а затем предложи исправления и proof-активы.
```

## 5. Создай FAQ в answer-ready формате для темы [Y]

```text
Создай FAQ в answer-ready формате для темы [Y]. Используй короткие прямые ответы, а затем детали и trust-сигналы.
```
