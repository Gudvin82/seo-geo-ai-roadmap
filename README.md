# SEO + GEO + AI Promotion Roadmap

Open-source framework for website growth across classic SEO, local GEO, AI visibility, and bilingual content operations.

[Русская версия](./README_RU.md)

## What is inside

- 25 roadmap sections in English and Russian
- Operational checklists for launch, content, local SEO, AI/GEO, and technical SEO
- Reusable templates for `llms.txt`, `robots.txt`, schema, briefs, KPI dashboards, and RACI
- Ready-to-use prompts for semantic research, AI audits, briefs, E-E-A-T reviews, and competitor analysis
- Starter Python scripts for AI bot access, `llms.txt`, sitemap, and schema checks

## Who this is for

- SEO consultants and in-house SEO teams
- Founders and CMOs building multilingual sites
- Agencies working with Google, Yandex, Bing, and AI search surfaces
- Content and GEO teams that want a repeatable operating system

## Quick Start

1. Read [docs/en/00-business-strategy.md](./docs/en/00-business-strategy.md).
2. Run the audit flow in [docs/en/01-audit.md](./docs/en/01-audit.md).
3. Prioritize AI visibility with [docs/en/08-geo-ai-search.md](./docs/en/08-geo-ai-search.md).
4. Copy the templates from [templates](./templates/).
5. Use the prompts from [prompts/en](./prompts/en/).
