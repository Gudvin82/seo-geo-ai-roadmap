# Positioning

## Mission

Build the first practical open-source operating manual for SEO, GEO, and AI discoverability across global and Russian-speaking markets.

## Core statement

Not another SEO checklist. An operating system for search, AI visibility, and multilingual discoverability.

## Russian adaptation

Не очередной чеклист по SEO, а рабочая система роста видимости сайта в поиске, AI-ответах и локальных платформах.

## Product framing

This repository is designed as a discoverability operating system:

- strategy first, not keyword chaos
- execution first, not theory first
- bilingual by design, not post-hoc translation
- governance aware, not content-only
- AI-native, not SEO-only

## What it is not

- not a crawler platform
- not a self-hosted AI chat UI
- not a rank tracker product
- not a single-market checklist

## Practical promise

A contributor should be able to open the repository, pick a section, run a checklist, adapt a template, use a prompt, validate with a script, and ship an improvement in one work session.
