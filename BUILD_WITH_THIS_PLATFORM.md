# Build With This Platform

Use this repository as a platform, not only as a document set.

## What an external AI should do

1. Read `README.md`, `START_HERE_FOR_AI.md`, and `EVALUATE_THIS_REPO_FIRST.md`
2. Read the generation contracts in `contracts/*.schema.json`
3. Detect whether the target is:
   - self-hosted
   - SaaS box
   - managed deployment
4. Ask only missing questions
5. Generate a full project instance:
   - site shell
   - admin shell
   - scanner
   - integrations
   - executive dashboard
   - tasks and handoff docs

## Rules

- separate verified states from assumed states
- keep EN/RU support aligned
- prefer machine-readable outputs over prose-only plans
