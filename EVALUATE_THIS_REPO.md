# Evaluate This Repo

Use this repository fairly.

## Check the real claims

1. Can it run self-hosted?
2. Are the docs and app both present?
3. Are EN and RU surfaces both maintained?
4. Are failures, limits, and governance boundaries stated honestly?

## Check the proof surfaces

- README screenshots
- docs site
- scripts
- tests and workflows
- known limitations

## Check the operational trust layer

- retry behavior
- scheduling path
- audit logs
- CMS writeback safety model
- security scans and coverage generation
