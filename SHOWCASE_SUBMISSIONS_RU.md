# Showcase Submissions

Используйте этот файл, если хотите понять, как выглядит сильная публичная
submission для репозитория.

## Хорошие типы submission

- bounded public case
- synthetic training case
- улучшение launch materials
- отчет о deployment friction
- улучшение script-first operator workflow

## Чеклист сильной submission

- четкая граница fact vs inference
- повторяемый operator lesson
- явный market context
- ссылки на proof assets или docs
- безопасная публичная формулировка
