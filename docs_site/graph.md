# Graph Intelligence

`v3.8.0` adds an explainable graph layer for discoverability operations.

Use it to visualize:

- site structure
- discoverability surfaces
- issue dependencies
- entity and trust relationships

App surface:

- `app/frontend/graph.html`

Canonical command:

- `/geo graph`

Recommended sequence:

1. `/geo audit`
2. `/geo graph`
3. `/geo report`
4. `/geo compare`
