# SaaS App Layer

The repository now includes a first product layer under `app/`.

What it adds:

- authentication
- workspaces
- projects and sites
- brand-facts truth center
- provider configurations
- structured audit runs
- reports and artifacts
- demo seed mode
- self-hosted hardening and security guidance

Core paths:

- `app/backend/`
- `app/frontend/`
- `app/shared/`
- `docker-compose.yml`
