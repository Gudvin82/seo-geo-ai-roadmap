# Deployment

The app layer can be run:

- locally with SQLite
- through Docker Compose
- as a self-hosted PostgreSQL-backed stack

Main entrypoints:

- `.env.example`
- `docker-compose.yml`
- `app/backend/Dockerfile`
- `app/frontend/Dockerfile`
